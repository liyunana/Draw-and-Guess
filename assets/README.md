# Draw & Guess - Assets 资源目录

## 目录结构

```
assets/
├── images/          # 图片资源
│   ├── icons/      # 图标
│   ├── ui/         # UI 元素
│   └── backgrounds/ # 背景图
└── sounds/          # 音效资源
    ├── effects/    # 音效
    └── music/      # 背景音乐
```

## 资源规范

### 图片资源
- 格式：PNG（透明背景），JPG（不透明背景）
- 图标：32x32, 64x64, 128x128
- UI 元素：根据需要设计
- 背景图：1280x720 或更高

### 音效资源
- 格式：WAV（高质量），MP3（压缩）
- 采样率：44100 Hz
- 比特率：128 kbps 以上

## 添加资源

1. 将资源文件放到对应目录
2. 在代码中使用相对路径引用
3. 提交时注意文件大小
